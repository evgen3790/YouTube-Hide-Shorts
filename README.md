# YouTube-Hide-Shorts
Скрывает раздел "Shorts" в меню YouTube (стандартном и мини). Работает только для Google Chrome. Простая установка, открытый код Идеально для тех, кто хочет чистый интерфейс YouTube без Shorts.

Инструкция по установке расширения YouTube-Hide-Shorts
Скачайте файлы (manifest.json, content.js, styles.css) из данного репозитория
Распакуйте архив в любую **папку** (например, YouTube Hide Shorts).

Добавьте расширение в Chrome: Откройте Chrome, введите в адресной строке chrome://extensions/ и нажмите Enter.
Включите Режим разработчика (переключатель в правом верхнем углу).
Нажмите Загрузить распакованное расширение.
Выберите папку YouTube Hide Shorts и нажмите Выбрать.

Обновите страницу (F5) и все будет работать.

**Примечание: Не удаляйте папку с файлами, иначе расширение перестанет работать.**
